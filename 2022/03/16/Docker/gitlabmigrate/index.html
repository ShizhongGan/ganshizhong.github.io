基于docker部署的gitlab迁移

2022-03-16 Docker docker, gitlab 0 评论

环境准备

准备一台新云主机，系统centos7.6，挂载一块硬盘（以往经验挺耗内存的）

搭建私人git仓库

更换远程仓库地址

首先进入项目所在文件夹，右键git bash

（1）查看当前的远程地址

      git remote -v

（2）删除当前的远程地址

     git remote rm origin

（3）添加远程地址

     git remote add origin (git@...)

（*4）首次推代码

     git push -u origin master
